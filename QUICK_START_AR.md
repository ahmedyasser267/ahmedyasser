# 🚀 دليل البدء السريع - موقع بورتفوليو أحمد ياسر

## ⚡ البدء في 3 دقائق

### 1️⃣ افتح المشروع

افتح `index.html` في المتصفح مباشرة، أو استخدم:

```bash
# Windows PowerShell
python -m http.server 8000

# ثم افتح المتصفح على:
# http://localhost:8000
```

### 2️⃣ عدّل المعلومات

افتح `index.html` وابحث عن:

- **السطر ~150**: Hero Section - عدّل الاسم والوصف
- **السطر ~200**: About Section - عدّل المعلومات الشخصية
- **السطر ~400**: Contact Section - عدّل معلومات الاتصال

### 3️⃣ غيّر الألوان (اختياري)

افتح `css/style.css` وعدّل:

```css
:root {
  --primary: #6366F1;      /* اللون الأساسي */
  --secondary: #EC4899;    /* اللون الثانوي */
  --accent: #14B8A6;      /* لون التمييز */
}
```

### 4️⃣ أضف مشاريعك

في قسم Projects، انسخ بطاقة مشروع وأضف مشروعك:

```html
<article class="project-card">
  <div class="project-content">
    <h3>اسم مشروعك</h3>
    <p>وصف المشروع...</p>
  </div>
</article>
```

## 📝 معلومات مهمة يجب تحديثها

### في `index.html`:

1. **Meta Tags** (السطر 9-17):
   - Title
   - Description
   - Keywords
   - Canonical URL

2. **Schema Markup** (السطر ~60):
   - الاسم الكامل
   - البريد الإلكتروني
   - رقم الهاتف
   - الروابط الاجتماعية

3. **Hero Section** (السطر ~150):
   - الاسم
   - الوصف
   - روابط السوشيال ميديا

4. **About Section** (السطر ~200):
   - المعلومات الشخصية
   - الإحصائيات

5. **Contact Section** (السطر ~400):
   - معلومات الاتصال

### في `sitemap.xml`:

- غيّر `https://ahmedyasser.dev/` إلى رابط موقعك

### في `robots.txt`:

- غيّر `https://ahmedyasser.dev/` إلى رابط موقعك

### في `manifest.json`:

- عدّل الاسم والوصف
- أضف الأيقونات في `assets/icons/`

## 🎨 إضافة الصور

1. ضع صورك في `assets/images/`
2. استخدم WebP format للسرعة
3. أضف alt text لكل صورة

## 🌐 النشر السريع

### Netlify (الأسهل):
1. اذهب إلى [netlify.com](https://netlify.com)
2. اسحب مجلد المشروع
3. انتهى! 🎉

### GitHub Pages:
1. ارفع المشروع على GitHub
2. Settings > Pages
3. اختر Branch الرئيسي
4. انتهى! 🎉
